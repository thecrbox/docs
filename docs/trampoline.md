# Welcome!

You're here because you're setting up your *thecrbox*. Great!

## Why?

To access all features (e.g., fine-tuning the PID if needed), *thecrbox* must be accessible via the Wi-Fi
network you plan to use for control. There’s no dedicated Android/iOS app -- you’ll enter your Wi-Fi
details during setup.

## Can it go wrong?

Wi-Fi names and passwords are case-sensitive.

If you enter the wrong credentials and *thecrbox* does not join your network, you can always
[force a factory reset][ui-factoryreset] and try again.
